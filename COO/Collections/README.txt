Conjunto de estruturas que implementam diversas estruturas de dados clássicas (listas, filas, pilhas, árvores...)
- ESTRUTURAS IMPLEMENTADAS DE FORMA MAIS EFICIENTE POSSÍVEL
- ARQUITERURA UNIFICADA PERMITE TROCA DA ESTRUTURA POR OUTRA
- ESTRUTURAS DO FRAMEWORK SÃO DINÂMICAS DINÂMICAS
- FUNÇÕES BÁSICAS ADICIONAR, REMOVER, VERIFICAR E PERCORRER
- É POSSÍVEL UTILIZAR VARIÁVEL <GENÉRICOS> DE QUALQUER TIPO (COM VERIFICAÇÃO)
Interfaces base:
	- Collections	Comportamentos básicos
	- List		Armazena de forma linear
	- Set		Não aceita elementos repetidos/ não define ordem
	- StoresSet	Elementos ordenados pelo valor

	- Map		Guarda valores e chaves
Classes bases:
	- ArrayList	Lista linear
	- LinkedList	Lista ligada
	- HashSet	Tabela de espalhamento
	- TreeSet	Árvore binária balanceada

	- HashMap	Chaves organizadas em Tabela de espalhamento
	- TreeMap	Chaves organizadas em Árvore de busca binária
